# 客服模块集成美洽SDK指南

## 项目概述

本项目为房间预约系统新增了客服模块，支持集成第三方美洽SDK平台，提供在线客服、消息记录等功能。

## 功能特性

### 前端功能
- ✅ 在线客服列表展示
- ✅ 实时聊天窗口
- ✅ 消息记录查询
- ✅ 客服管理（管理员）
- ✅ 客服启用/禁用
- ✅ 响应式设计

### 后端功能
- ✅ 客服信息管理
- ✅ 消息记录存储
- ✅ 美洽SDK集成接口
- ✅ 权限控制

## 后端集成步骤

### 1. 数据库配置

执行 `customer_service.sql` 脚本创建相关表：

```sql

-- 创建客服消息表
CREATE TABLE customer_message (...)
```

### 2. 配置美洽SDK

在 `application.yml` 中添加美洽配置：

```yaml
meiqia:
  enabled: true
  enterprise-id: your_enterprise_id
  api-key: your_api_key
```

## API接口文档

### 客服相关接口

#### 获取客服列表
```
POST /api/customerService/list
请求体：{ page: 1, limit: 10 }
```

#### 获取启用的客服
```
POST /api/customerService/getEnabledServices
请求体：{ page: 1, limit: 10 }
```

#### 创建或修改客服
```
POST /api/customerService/createOrEdit
请求体：{
  id: 1,
  serviceName: "客服名称",
  serviceDescription: "描述",
  avatar: "头像URL",
  enabled: 1
}
```

#### 删除客服
```
POST /api/customerService/delete
请求体：{ id: 1 }
```

### 消息相关接口

#### 发送消息
```
POST /api/customerMessage/sendMessage
请求体：{
  serviceId: 1,
  messageContent: "消息内容"
}
```

#### 获取聊天记录
```
GET /api/customerMessage/getChatHistory?userId=1&serviceId=1&page=1&limit=10
```

#### 获取消息列表
```
POST /api/customerMessage/list
请求体：{ page: 1, limit: 10, userId: 1, serviceId: 1 }
```

## 美洽SDK集成

### 获取美洽账户信息

1. 访问 [美洽官网](https://www.meiqia.com/)
2. 注册企业账户
3. 获取企业ID和API密钥
4. 在后端配置中填入相关信息

### 美洽SDK配置

`MeiqiaConfig.java` 配置类：

```java
@Data
@Component
@ConfigurationProperties(prefix = "meiqia")
public class MeiqiaConfig {
    private String enterpriseId;      // 企业ID
    private String apiKey;            // API密钥
    private String apiBaseUrl;        // API基础URL
    private Boolean enabled;          // 是否启用
}
```

### 美洽服务集成

`MeiqiaService.java` 提供以下方法：

- `initMeiqia()` - 初始化美洽SDK
- `createService()` - 创建客服
- `sendMessage()` - 发送消息
- `getServiceStatus()` - 获取客服状态

## 使用示例

### 前端使用

```javascript
// 获取启用的客服列表
import { getEnabledServices, sendMessage } from '@/api/customerService'

// 获取客服列表
getEnabledServices({ page: 1, limit: 10 }).then(res => {
  console.log(res.data.items)
})

// 发送消息
sendMessage({
  serviceId: 1,
  messageContent: '您好，我需要帮助'
}).then(res => {
  console.log('消息已发送')
})
```

## 注意事项

1. **权限控制**：客服管理功能仅限管理员访问
2. **消息存储**：所有消息都会存储在数据库中，便于查询和统计
3. **美洽集成**：需要在美洽平台创建企业账户并获取相关凭证
4. **头像URL**：建议使用CDN或对象存储服务存储头像
5. **消息加密**：生产环境建议对敏感消息进行加密存储

## 常见问题

### Q: 如何启用美洽SDK？
A: 在 `application.yml` 中设置 `meiqia.enabled: true`，并填入企业ID和API密钥。

### Q: 消息是否实时同步到美洽？
A: 当前实现中消息存储在本地数据库。如需实时同步，需要在 `MeiqiaService` 中调用美洽API。

### Q: 如何修改客服头像？
A: 在客服管理页面编辑客服信息，填入头像URL即可。

### Q: 支持多客服转接吗？
A: 当前版本不支持，可在后续版本中添加此功能。
