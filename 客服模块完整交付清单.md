## 数据库表结构

### customer_service 表

```sql
CREATE TABLE customer_service (
  id INT PRIMARY KEY AUTO_INCREMENT,
  service_name VARCHAR(100) NOT NULL,
  service_description VARCHAR(500),
  avatar VARCHAR(500),
  status INT DEFAULT 0,
  meiqia_service_id VARCHAR(100),
  enabled INT DEFAULT 1,
  creation_time DATETIME DEFAULT CURRENT_TIMESTAMP,
  update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
)
```

**字段说明**
- id: 主键
- service_name: 客服名称
- service_description: 客服描述
- avatar: 客服头像URL
- status: 客服状态（0-离线，1-在线）
- meiqia_service_id: 美洽客服ID
- enabled: 是否启用（0-禁用，1-启用）
- creation_time: 创建时间
- update_time: 更新时间

### customer_message 表

```sql
CREATE TABLE customer_message (
  id INT PRIMARY KEY AUTO_INCREMENT,
  user_id INT NOT NULL,
  service_id INT NOT NULL,
  message_content LONGTEXT NOT NULL,
  message_type INT DEFAULT 0,
  meiqia_message_id VARCHAR(100),
  is_read INT DEFAULT 0,
  creation_time DATETIME DEFAULT CURRENT_TIMESTAMP,
  update_time DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  FOREIGN KEY (user_id) REFERENCES app_user(id),
  FOREIGN KEY (service_id) REFERENCES customer_service(id)
)
```

**字段说明**
- id: 主键
- user_id: 用户ID
- service_id: 客服ID
- message_content: 消息内容
- message_type: 消息类型（0-用户，1-客服）
- meiqia_message_id: 美洽消息ID
- is_read: 是否已读（0-未读，1-已读）
- creation_time: 创建时间
- update_time: 更新时间

## 集成步骤

### 第一步：数据库集成

```bash
# 执行SQL脚本
mysql -u root -p < customer_service.sql
```

### 第二步：后端集成

1. 复制所有Java文件到对应目录
2. 在 application.yml 中配置美洽参数
3. 重启Spring Boot应用

### 第三步：前端集成

1. 复制 customerService.js 到 src/api/
2. 复制 CustomerService.vue 到 src/views/
3. 路由已自动配置
4. 重启Vue开发服务器

### 第四步：美洽配置

1. 获取美洽企业ID和API密钥
2. 在 application.yml 中配置
3. 启用美洽服务
