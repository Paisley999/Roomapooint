# 客服模块快速开始

## 5分钟快速集成

### 第一步：执行数据库脚本

```sql
-- 在MySQL中执行 customer_service.sql 文件
-- 创建 customer_service 和 customer_message 表
```

### 第二步：配置后端

在 `application.yml` 中添加美洽配置：

```yaml
meiqia:
  enabled: true
  enterprise-id: your_enterprise_id  # 替换为你的企业ID
  api-key: your_api_key              # 替换为你的API密钥
  api-base-url: https://api.meiqia.com
```

### 第三步：启动应用

```bash
# 后端启动
mvn spring-boot:run

# 前端启动
npm run serve
```

### 第四步：访问客服模块

- **用户端**：http://localhost:8080/#/Front/CustomerService
- **管理端**：http://localhost:8080/#/Admin/CustomerService

## 核心功能

### 用户端功能

1. **查看在线客服**
   - 显示所有启用的客服列表
   - 显示客服在线状态

2. **与客服聊天**
   - 选择客服开始聊天
   - 发送和接收消息
   - 查看聊天历史

### 管理端功能

1. **客服管理**
   - 添加新客服
   - 编辑客服信息
   - 删除客服
   - 启用/禁用客服

2. **消息管理**
   - 查看所有消息
   - 消息统计

## API端点

### 客服接口

```
GET  /api/customerService/list              # 获取客服列表
POST /api/customerService/getEnabledServices # 获取启用的客服
POST /api/customerService/createOrEdit      # 创建/编辑客服
POST /api/customerService/delete            # 删除客服
```

### 消息接口

```
POST /api/customerMessage/sendMessage       # 发送消息
GET  /api/customerMessage/getChatHistory    # 获取聊天记录
POST /api/customerMessage/list              # 获取消息列表
```

### 调用客服API

```javascript
import { 
  getEnabledServices, 
  sendMessage, 
  getChatHistory 
} from '@/api/customerService'

// 获取启用的客服
getEnabledServices({ page: 1, limit: 10 }).then(res => {
  console.log(res.data.items)
})

// 发送消息
sendMessage({
  serviceId: 1,
  messageContent: '您好'
}).then(res => {
  console.log('消息已发送')
})

// 获取聊天记录
getChatHistory(userId, serviceId, 1, 10).then(res => {
  console.log(res.data.items)
})
```

## 常见问题

### Q: 如何测试客服功能？
A: 
1. 以用户身份登录，访问 `/Front/CustomerService`
2. 以管理员身份登录，访问 `/Admin/CustomerService` 添加客服
3. 用户端选择客服进行聊天

### Q: 消息是否实时同步？
A: 当前版本消息存储在本地数据库。实时同步需要集成WebSocket或美洽实时API。

### Q: 如何修改客服头像？
A: 在管理端编辑客服信息，填入头像URL即可。

### Q: 支持离线消息吗？
A: 支持。用户可以给离线客服发送消息，客服上线后可以查看。
