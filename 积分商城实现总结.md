## 📊 数据流

```
用户登录
    ↓
进入积分商城
    ↓
获取用户积分 ← 调用 /Integral/GetMyIntegralData
    ↓
获取商品列表 ← 调用 /IntegralMall/GetProductList
    ↓
浏览商品
    ↓
选择商品点击兑换
    ↓
弹出确认对话框
    ↓
用户确认
    ↓
调用 /IntegralMall/BuyProduct
    ↓
后端验证：
  - 用户积分是否充足
  - 商品是否存在
  - 商品是否上架
  - 库存是否充足
    ↓
扣除用户积分
    ↓
更新商品库存
    ↓
记录兑换历史
    ↓
返回成功信息
    ↓
前端更新积分和商品列表
    ↓
显示成功提示
```

## 💡 关键实现细节

### 1. 积分验证
```javascript
// 前端验证
if (userIntegral < product.integralPrice) {
  // 禁用兑换按钮
  // 显示"积分不足"提示
}

// 后端验证
if (userIntegral.getTotalIntegral() < totalIntegral) {
  throw new BusinessException("积分不足");
}
```

### 2. 库存管理
```javascript
// 检查库存
if (product.stock < quantity) {
  throw new BusinessException("商品库存不足");
}

// 更新库存
product.setStock(product.getStock() - quantity);
```

### 3. 积分扣除
```javascript
// 扣除积分
integralService.deductIntegral(userId, totalIntegral, 
  "兑换商品：" + product.getProductName(), 
  String.valueOf(productId));
```
