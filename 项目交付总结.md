## 快速开始

### 1. 数据库集成
```bash
mysql -u root -p < customer_service.sql
```

### 2. 后端配置
在 `application.yml` 中添加：
```yaml
meiqia:
  enabled: true
  enterprise-id: your_enterprise_id
  api-key: your_api_key
  api-base-url: https://api.meiqia.com
```

### 3. 启动应用
```bash
# 后端
mvn spring-boot:run

# 前端
npm run serve
```

### 4. 访问客服模块
- 用户端：http://localhost:8080/#/Front/CustomerService
- 管理端：http://localhost:8080/#/Admin/CustomerService

## 常见问题

### Q: 如何启用美洽SDK？
A: 在application.yml中设置meiqia.enabled: true，并填入企业ID和API密钥。

### Q: 消息是否实时同步到美洽？
A: 当前实现中消息存储在本地数据库。可通过异步任务定期同步到美洽。

### Q: 如何修改客服头像？
A: 在客服管理页面编辑客服信息，填入头像URL即可。

### Q: 支持多客服转接吗？
A: 当前版本不支持，可在后续版本中添加此功能。

### Q: 如何处理美洽API超时？
A: 实现重试机制和超时处理，使用异步处理避免阻塞。
